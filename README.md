# WanAndroid
# MVP+RxJava+Retrofit+Arouter+模块化 的练手项目
* 1、为何写这个项目：
> 在此前MVP分支基础上，实现模块化开发，目的是走一边模块化开发的流程，熟悉各个步骤要做的事及模块化开发过程中遇到的坑及解决办法

* 2、通过这个项目我们能学到什么：   
> a、了解模块化开发大致流程   
b、了解到ARouter在跨模块中的用法  
c、了解模块化开发会遇到哪些问题  
d、动手实现适合自己的模块化开发架构
